# 🔐 CipherVault

A modern **Flask-based Encryption & Security Toolkit** that enables secure text encryption, file encryption, cryptographic hashing, key generation, password management, and PDF security reporting through an interactive cybersecurity dashboard.

## ✨ Features

### 🔒 Encryption
- Caesar Cipher
- ROT13 Cipher
- XOR Cipher
- Base64 Encoding

### 📁 File Security
- Encrypt Files
- Decrypt Files
- Secure File Download

### 🔑 Security Tools
- Secure Random Key Generator
- Cryptographic Hash Generator
  - SHA-1
  - SHA-256
  - SHA-384
  - SHA-512

### 🛡 Additional Features
- Password Manager
- Interactive Dashboard
- Activity History
- Download Center
- PDF Security Report Generator
- Responsive Dark UI

---

## 🛠 Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML5, CSS3, JavaScript
- **PDF Generation:** ReportLab
- **Storage:** Browser LocalStorage

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Purushothama449/CipherVault.git
```

### Navigate to the project

```bash
cd CipherVault
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

### Open your browser

```
http://127.0.0.1:5000
```
