About Cyber Encryption Center

Version 1.0.0 · Built with Flask

Cyber Encryption Center is a lightweight toolkit for learning and experimenting with classic encryption techniques: Caesar Cipher, ROT13, Base64 encoding, and XOR ciphers, alongside hash generation, key generation, file encryption, and a local password vault.

These are educational ciphers, not production-grade cryptography — avoid using them to protect anything genuinely sensitive.
